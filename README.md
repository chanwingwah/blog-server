# myblog-node 个人博客服务端

## 介绍
个人博客网站后端实现， 博客地址[陈永华的博客](http://chanwingwah.info/)
服务端语言使用的是node.js , 数据库使用的是mongodb

## 实现功能

- [ ] Koa2框架设计路由
- [ ] mongoose 连接mongoDB 数据存储
- [ ] Redis 存储数据库 存储session
- [ ] 开发环境 nodemon 监听服务
- [ ] 生存环境 PM2 进程管理
- [ ] 用户登录验证 cookie session
- [ ] 用户登录验证 cookie session
- [ ] 博客增删查改
- [ ] 行博增删查改
- [ ] 博客评论增删查改
- [ ] 书单增删查改
